# Discord-Jellyfin-Stream
Stream Jellyfin 24/7 + Recovery 

Controller Service
   ├ Jellyfin API integration
   ├ Discord bot interface
   ├ Playback state manager
   └ Queue / permissions

Streaming Node
   ├ Headless browser
   ├ OBS or WebRTC injector
   ├ Hardware encoding
   └ Session watchdog

User Layer
   ├ Discord commands
   ├ Voting / playlists
   └ Sync state
